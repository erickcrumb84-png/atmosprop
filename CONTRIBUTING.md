# Contributing to Atmosprop

Thank you for considering contributing to Atmosprop! We're excited to welcome contributors who are interested in improving the project. Before you start, please take a moment to read through the guidelines outlined below.

## How to Contribute
1. **Fork the Repository**: Use the "Fork" button in the top right corner to create your own copy of the repository.
2. **Create a Branch**: Create a new branch for your feature or fix. You can name it something like `feature/my-awesome-feature` or `bugfix/fix-some-bug`.
3. **Make Your Changes**: Make the necessary changes to improve the project. Ensure that your code adheres to the existing coding style.
4. **Submit a Pull Request**: After making your changes, submit a pull request. Describe what you changed and why. Your pull request will be reviewed by the maintainers.

## Code of Conduct
By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md). Please report any incidents to the maintainers.

Happy coding!
